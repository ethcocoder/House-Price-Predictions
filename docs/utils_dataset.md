# Massive Dataset Utility

`src/utils/dataset.py`

This utility provides methods to fetch large-scale housing data and synthetically expand existing datasets for production stress testing.

## Features
- **OpenML Integration**: Fetches the King County House Sales dataset (21k records).
- **Synthetic Augmentation**: Can expand any numeric dataset to a target size (e.g., 250k+ rows) using noise-injected replication. This maintains statistical distributions while providing volume for deep learning.
- **Scikit-Learn Integration**: Includes a fetcher for the California Housing dataset.

## Usage
To generate a massive 250,000-row dataset:
```bash
python -m src.utils.dataset
```

This will save `massive_housing_data.csv` to the `data/raw/` directory.

## Importance for Deep Learning
Neural Networks (like the one in `src/models/deep_learning.py`) perform significantly better as data volume increases. Using the massive dataset generated by this utility will help the DL model converge on more robust patterns and potentially outperform Linear Regression.
